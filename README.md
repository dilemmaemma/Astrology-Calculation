# Natal Chart Calculation

## Uses
Used to find natal chart positions of your personal chart, and will later allow for the implementation of composite charts

## Includes
Planets included:
- Sun
- Moon
- Mercury
- Venus
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune
- Pluto
Asteroids included:
- Ceres
- Chiron
- Juno
- Lilith
- Pallas
- Pholus
- Vesta
Angles/points included:
- Ascendent
- Descendent
- Midheaven
- Imum Coeli
- Vertex

## Needed to use
- Birth time
- Birth location
- Birth day, month, and year
